Project Summary

The purpose of this project is to analyze how weather changes as people get closer to the equator.We pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather using latitude. Factors we looked are included: temperature, cloudiness, wind speed, and humidity. This site provices the surce data and visualizations created as part of analysis, as well as explanations and descriptions of any trenda and correlations witnessed.
