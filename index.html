Project Summary


The purpose of this project is to analyze how weather changes as people get closer to the equator. We pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather using latitude. The factors that we looked at are temperature, cloudiness, wind speed, and humidity. This site provides the source data, visualizations created as part of the analysis, explanations, and descriptions of any trends and correlations witnessed.
